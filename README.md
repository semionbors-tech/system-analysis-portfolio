# system-analysis-portfolio
Модуль 4. Продуктовый анализ логов (Python, Pandas, NumPy)Для проверки стабильности спроектированного API авторизации был разработан скрипт автоматического анализа серверных логов.Что делает скрипт:
Импортирует сырые данные логов посещаемости.Проводит очистку данных (обработка Null-значений средствами Pandas) [hi-tech.mail.ru].Использует векторизацию NumPy для быстрой классификации статусов ответов (Success/Error) [hi-tech.mail.ru].Рассчитывает метрики уникальных пользователей (nunique) и распределение трафика по регионам (groupby).
